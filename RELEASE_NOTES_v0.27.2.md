# ZEKE v0.27.2 — Exercise Screen Mockup Fidelity

Implements the user-supplied exercise-entry mockup as the controlling specification. The selected exercise replaces the workout list on screen. The hierarchy is: minimal header, Coach’s Eye, progression, last time, today’s entry, explicit Save Exercise feedback, and an in-place Form Guide bottom sheet over the dimmed exercise screen. Vertical scrolling is allowed; horizontal page scrolling is prohibited.
