# ZEKE v0.8.1 Evaluation Alpha

Build: **2026.07.11.2**

## Purpose

This release is designed for immediate product feedback using the user's connected/imported ZEKE records. It does not add demo data and does not change the repository, storage, interpretation, or AI-routing architecture.

## Visible changes

- Added an evidence-coverage strip showing verified records, tracked metrics, workout sessions, lab observations, freshest evidence, and unresolved clarifications.
- Rebuilt **I've been thinking…** as a compact synthesis feed drawing from discoveries, Coach's Eye, current verified metrics, and recurring signals in the user's own record.
- Strengthened dashboard information hierarchy, card depth, metric emphasis, and mobile behavior.
- Preserved the unified conversation, Knowledge Inbox/event repository, confirmation rules, Google Drive adapter, and provider-independent AI Router.
- Continued to hide unavailable metrics and avoid interpolation or invented personal data.

## Evaluation focus

Assess whether the dashboard surfaces the right information, whether its hierarchy feels useful, and which data-derived observations should be promoted, demoted, or changed in the next release.
