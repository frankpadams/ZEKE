# ZEKE Data Dictionary

## Event

Something that happened, was planned, was observed, or was recorded.

Fields:
- event_id
- chapter
- event_type
- event_time
- event_time_precision
- recorded_time
- modified_time
- title
- summary
- raw_input
- status
- confidence
- source
- review_status
- privacy_class
- related_entities
- relationships
- notes

## Entity

A person, object, place, medication, vehicle, exercise, provider, document, or concept.

## Relationship

A connection between events or entities.

Relationship types include:
- observed
- associated
- user-linked
- clinician-linked
- evidence-supported
- hypothesis
- historical-context

## Observation

Something reported or measured.

## Guidance

A suggestion for consideration. Guidance is not a command.

## Discovery Project

A user-approved focused investigation.
