# ZEKE Vision

## Mission

ZEKE exists to help people understand, preserve, and write their own story.

## The problem

Personal data is scattered across apps, documents, calendars, devices, photos, screenshots, medical portals, spreadsheets, and memories. Most tools capture fragments. Few preserve context.

## The opportunity

ZEKE can become a personal context engine that helps users answer:

1. What happened?
2. What does it mean?
3. How does it fit into my broader story?
4. What can I learn from it?
5. What should I consider doing next?

## Initial chapter

ZEKE begins with Health because health data is important, longitudinal, fragmented, and highly contextual.

## Signature features

- Tell ZEKE
- Review before permanent save
- Story timeline
- Patterns I've Noticed
- Discovery Projects
- Conversational dashboard customization
- Domain-specific privacy
- AI-agnostic reasoning
- Explain and contextualize
