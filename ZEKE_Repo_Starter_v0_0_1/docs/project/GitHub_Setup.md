# GitHub Setup

Repository:

https://github.com/frankpadams/ZEKE

## First upload

1. Download the ZEKE starter ZIP.
2. Unzip it.
3. Open the GitHub repository.
4. Choose **Add file** → **Upload files**.
5. Drag the unzipped files and folders into GitHub.
6. Commit with:

```text
Initial ZEKE project foundation: vision, constitution, architecture, roadmap, and prototype framework
```

Do not add API keys or secrets to the repository.
