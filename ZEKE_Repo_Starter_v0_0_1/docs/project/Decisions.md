# ZEKE Decision Log

## Decision #001 — ZEKE name

ZEKE means Zero-friction Evidence & Knowledge Engine.

## Decision #002 — Mission

ZEKE exists to help people understand, preserve, and write their own story.

## Decision #003 — AI is a consultant

AI providers are tools ZEKE can consult, not the owner of the Story.

## Decision #004 — Review before permanent save

Meaningful permanent Story changes require user approval.

## Decision #005 — Historical context icon

Use 🕰️ for historical context.

## Decision #006 — Guidance icon

Use 🧭 for guidance.

## Decision #007 — Scheduled is not completed

Planned, scheduled, or expected events are not treated as completed without confirmation.

## Decision #008 — Negation overrides keywords

The parser must detect negation before keyword-based event detection.

## Decision #009 — Domain-specific privacy

Privacy rules differ by chapter/domain.

## Decision #010 — $0/month AI budget

Default AI budget is $0/month.

## Decision #011 — Manual AI packages are fallback

Manual packages remain available but should not be the primary daily workflow.

## Decision #012 — GitHub repository

ZEKE will live in a GitHub repository for version history, planning, and documentation.
