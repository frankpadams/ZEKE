# ZEKE Integration Catalog

## Philosophy

Integrations reduce manual work. They are tools, not dependencies.

## Priority integrations

### Google Sheets / Drive

Status: early prototype path  
Cost: free  
Purpose: live human-readable storage workspace

### Free AI provider

Status: planned early  
Cost: $0/month target  
Purpose: live answers without manual package upload

### Google Calendar

Status: planned early  
Cost: free  
Purpose: scheduled context and reminders  
Rule: calendar event is not proof of completion

### Apple Health

Status: future iOS integration  
Cost: free through HealthKit  
Purpose: objective health data  
Limitation: requires iOS app or companion app

### MyFitnessPal

Paths:
- MyFitnessPal to Apple Health to ZEKE
- Direct API later if approved and practical

### MyChart / Epic

Paths:
1. Manual PDF/screenshot import
2. Apple Health Records bridge
3. Direct Epic on FHIR later

### Planet Fitness

Paths:
- Manual screenshot/import of check-in history
- No public live API assumed

### Screenshot / PDF import

High priority. Extracted data requires review before permanent save.
