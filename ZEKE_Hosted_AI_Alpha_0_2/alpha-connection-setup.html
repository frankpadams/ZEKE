Project Zeke Alpha Connection Setup

This helper creates the one public configuration file needed for the alpha site to show the real Google sign-in window. It does not ask for or store your Google password, access token, refresh token, or personal data.

First complete the one-time Google application registration in ALPHA_GOOGLE_CONNECTION.md. Google will give you a Web Client ID ending in .apps.googleusercontent.com.

After downloading, replace the existing zeke-config.js file in the deployed Zeke site with the downloaded file. Then reload Zeke and use Connect Google Account.
